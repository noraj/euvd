# Usage

## Library

See examples in [the library documentation](https://noraj.github.io/euvd/yard/EUVD).

- [EUVD::API::Downloads](https://noraj.github.io/euvd/yard/EUVD/API/Downloads) - Access to bulk data download endpoints.
- [EUVD::API::Meta](https://noraj.github.io/euvd/yard/EUVD/API/Meta) - Access to meta/utility endpoints (banner, assigners list).
- [EUVD::API::Observations](https://noraj.github.io/euvd/yard/EUVD/API/Observations) - Access to honeypot observation and KEV entry endpoints.
- [EUVD::API::Records](https://noraj.github.io/euvd/yard/EUVD/API/Records) - Access to specific resource endpoints: EUVD record by ID and advisory by ID.
- [EUVD::API::Vulnerabilities](https://noraj.github.io/euvd/yard/EUVD/API/Vulnerabilities) - Access to vulnerability endpoints: latest, critical, exploited, and search.
