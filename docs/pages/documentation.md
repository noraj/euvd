# Documentation

### Serve locally

```bash
npm i docsify-cli gulp-cli -g
cd docs-tools
npm i
gulp
docsify serve ../docs
```

## Library doc

The output directory of the library documentation will be `docs/yard`.

You can consult it online [here](https://noraj.github.io/euvd/yard/).

### Build & serve locally

```bash
bundle exec yard doc && bundle exec yard server
```
