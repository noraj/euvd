GET /api/search
Query vulnerabilities with flexible filters.

Parameters:

  • params (Hash) (defaults to: {}) —

    Query parameters

Options Hash (params):

  • :text (String) —

    Keywords (searches across description, EUVD ID, aliases, product, vendor)

  • :fromScore (Float) —

    Minimum CVSS score (0-10)

  • :toScore (Float) —

    Maximum CVSS score (0-10)

  • :fromEpss (Integer) —

    Minimum EPSS score (0-100)

  • :toEpss (Integer) —

    Maximum EPSS score (0-100)

  • :fromDate (String) —

    Minimum published date (YYYY-MM-DD)

  • :toDate (String) —

    Maximum published date (YYYY-MM-DD)

  • :fromUpdatedDate (String) —

    Minimum updated date (YYYY-MM-DD)

  • :toUpdatedDate (String) —

    Maximum updated date (YYYY-MM-DD)

  • :product (String) —

    Product name

  • :vendor (String) —

    Vendor name

  • :assigner (String) —

    Assigner name

  • :exploited (Boolean) —

    Filter by exploited status

  • :page (Integer) —

    Page number (starts at 0)

  • :size (Integer) —

    Page size (default 10, max 100)

Returns:

  • (Sawyer::Resource) —

    Search results with +.items+ (Array) and +.total+ (Integer)
